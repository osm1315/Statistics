# 11.데이터 전처리와 파생변수 생성

## 11.1. 결측값 처리
- 완전 무작위 결측: 결측값이 무작위로 발생한 경우 -> 제거해도 편향이 거의 발생되지 않음
- 무작위 결측: 다른 변수의 특성에 의해 해당 변수의 결측치가 체계적으로 발생한 경우
    - 예를 들어 전국 체인 매출정보 중 특정 체인점의 포스기 오류로 해당 체인점에 해당하는 매출 정보에 결측값이 많이 나타는 경우
- 비무작위 결측: 결측값들이 해당 변수 자체의 특성을 갖고 있는 경우

결측값 처리 방법
- 표본제거 방법: 결측값이 심하게 많은 변수를 제거하거나 결측값이 포함된 행을 제외하고 데이터 분석을 실행
    - 결측값 비율이 10% 미만일 경우
- 평균 대치법
    - 최빈값, 중앙값, 최댓값, 최솟값 대치도 가능
    - 사용은 간단하지만 표준오차가 왜곡되고 p-value가 부정확하게 되는 단점
    - 완전 무작위 결측이 아닌 경우 적절하지 않음
- 보간법: 데이터가 시계열적 특성을 가지고 있을 경우
- 회귀 대치법: 해당 변수와 다른 변수 사이의 관계성을 고려하여 결측값 계산
    - 예를 들어 연령 변수 결측값 대치에 연수입 변수 사용
    - 인위적으로 회귀식에 확률 오차항을 추가하는 확률적 회귀대치법도 있음
- 다중대치법: 지금까지의 단순 대치법들의 표본오차 과소추정문제를 해결하기 위한 방법, 단순 대치를 여러번 수행해 n개의 가상적 데이터를 생성하여 평균으로 결측값을 대치
    - 대치 단계 -> 분석 단계 -> 결합 단계
    ![다중대치법](./images/study5/스크린샷%202025-05-09%20오후%206.23.41.png)

## 11.2. 이상치 처리
이상치: 일부 관측치의 값이 전체 데이터의 범위에서 크게 벗어난 아주 작거나 큰 극단적인 값을 갖는 것

-> 분석이나 모델링의 정확도를 감소시킴

처리방법
- 결측치로 다뤄 결측치 처리를 하거나 제거
- 관측값 변경: 하한값과 상한값 결정후 대체
- 가중치 조정: 이상치의 영향을 감소시키는 가중치를 줌

도메인에 따라 적절한 방법으로 이상치 판단 후 처리하기

## 11.3. 변수 구간화
이산형 변수를 변주형 변수로 변환

-> 비즈니스적 상황에 맞도록 변환시켜 데이터의 해석이나 예측, 분류 모델을 의도에 맞도록 유도

## 11.4. 데이터 표준화와 정규화 스케일링
독립변수들이 서로 단위가 다르거나 편차가 심할때 값의 스케일을 일정한 수준으로 변환시켜줌

표준화: 각 관측치의 값이 전체 평균을 기준으로 어느정도 떨어져있는지 나타낼때 사용
- 평균은 0, 1표준편차거리는 +-1, 2표준편차 거리는 +-2로 변환됨
- zero-mean으로부터 얼마나 떨어져 있는지를 나타내기 때문에 z-score라 표현

정규화: 데이터의 범위를 0부터 1까지로 변환하여 데이터 분포를 조정

표준화와 정규화는 이상치에 민감함 -> robust scaler 사용

robust scaler: 데이터의 중앙값을 0으로 잡고 1사분위값과 3사분위값의 차이를 1이 되도록 함
- 이상치의 영향력을 최소화 

## 11.5. 모델 성능 향상을 위한 파생 변수 생성
파생변수: 원래 있던 변수들을 조합하거나 함수를 적용하여 새로 만들어낸 변수
- 데이터 구간화, 표준화 및 정규화
- 기존 값에 로그나 제곱근 등
- 과거 시점 대비 변화 정도 등

데이터의 특성을 이용하여 분석 효율을 높이는 것이기 때문에 
- 전체 데이터에 대한 파악
- 해당 비즈니스 도메인에 대한 충분한 이해

## 문제 1. 데이터 전처리

> **🧚 한 금융회사의 대출 데이터에서 `소득` 변수에 결측치가 포함되어 있다. 다음 중 가장 적절한 결측치 처리 방법은 무엇인가?**

> **[보기]   
1️⃣ 결측값이 포함된 행을 모두 제거한다.  
2️⃣ 결측값을 `소득` 변수의 평균값으로 대체한다.  
3️⃣ `연령`과 `직업군`을 독립변수로 사용하여 회귀 모델을 만들어 `소득` 값을 예측한다.  
4️⃣ 결측값을 보간법을 이용해 채운다.**

> **[데이터 특징]**     
    - `소득` 변수는 연속형 변수이다.  
    - 소득과 `연령`, `직업군` 간에 강한 상관관계가 있다.  
    - 데이터셋에서 `소득` 변수의 결측 비율은 15%이다.

```
결측 비율이 작지 않기 때문에 삭제하지 않고, 연령과 직업군 간 강한 상관관계가 있으므로 3번 회귀 대치법으로 처리한다
```
